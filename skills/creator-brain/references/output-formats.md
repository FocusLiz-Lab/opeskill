# Output Formats

Choose the smallest format that fits the task.

## Default Format

Use for mixed or general requests:

1. 结论
2. 选择的模块
3. 综合诊断
4. 行动方案
5. 风险与取舍
6. 下一步

## Commercialization Format

Use for `ahs`-led answers:

1. 商业诊断
2. Offer/定位建议
3. 获客路径
4. 成交或转化改进
5. 交付与运营要求
6. 7 天或 30 天执行计划

## Creator Business Format

Use for `dks`-led answers:

1. 定位
2. 受众
3. 内容系统
4. 产品阶梯
5. 变现路径
6. 每周执行节奏

## Organization Execution Format

Use for `lhs`-led answers:

1. 执行瓶颈
2. 角色与责任
3. SOP 或流程
4. 会议与检查节奏
5. 指标与问责
6. 30 天修复计划

## Wealth Judgment Format

Use for `nrs`-led answers:

1. 决策问题
2. 杠杆分析
3. 机会成本
4. 长期复利判断
5. 当前约束
6. 推荐选择和验证动作

## Health Performance Format

Use for `hbs`-led answers:

1. 可能瓶颈
2. 协议组合
3. 日常节奏
4. 追踪指标
5. 注意事项
6. 调整计划

## Behavior Change Format

Use for `mrs`-led answers:

1. 真正卡点
2. 立即行动触发器
3. 环境设计
4. 情绪或自信处理
5. 7 天行为计划
6. 失败恢复规则

## Trading Format

Use for `rts`-led answers:

1. 策略或市场背景
2. Setup 定义
3. 入场逻辑
4. 出场逻辑
5. 风险管理
6. 复盘流程
