# Dankoe 一人公司框架

## 核心循环

一人公司不是一个人做所有事，而是一个小型价值系统：

```text
身份/兴趣 -> 问题 -> 内容证明 -> offer -> 反馈 -> 更好的产品
```

AI 和工具只能加速这个循环，不能替代判断、品味、证明和重复。

## 方向判断

先审计用户的：

- 技能：能为别人做什么
- 兴趣：能长期研究什么
- 问题：过去反复解决过什么
- 证明：作品、结果、用户、收入、案例
- 约束：时间、平台、精力、风险

没有证明时，先设计 proof-building path，不要包装权威。

## 推荐路径

- 无受众但有技能：先做服务
- 有诊断能力：咨询/陪跑
- 有重复方法：数字产品
- 有持续表达能力：Newsletter / 社群
- 想现金和资产兼得：服务先行，再产品化

## 90 天节奏

- Days 1-14：定位、观点、初始证明
- Days 15-30：第一个 offer，用户对话，试点成交
- Days 31-60：内容围绕买家问题，交付沉淀 SOP
- Days 61-90：产品化、案例化、工作流化
