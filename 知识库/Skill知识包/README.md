# Skill 知识包

本目录收录 opeskill 使用的 7 位专家方法论知识包。

| 目录 | 专家 | 主要用途 |
|---|---|---|
| `ahs/` | Alex Hormozi | offer、定价、获客、销售、商业化 |
| `lhs/` | Leila Hormozi | 招聘、管理、运营、SOP、团队执行 |
| `dks/` | Dan Koe | 一人公司、个人品牌、内容系统、数字产品 |
| `nrs/` | Naval Ravikant | 财富判断、杠杆、长期主义、特定知识 |
| `hbs/` | Huberman Lab | 睡眠、专注、压力、精力、习惯 |
| `mrs/` | Mel Robbins | 行动力、拖延、自信、情绪阻力、习惯启动 |
| `rts/` | Rayner Teo | 交易系统、趋势、价格行为、风控、复盘 |

每个知识包可单独阅读，也可配合 `知识库/原子库/{专家代码}/atoms.jsonl` 做检索增强。
