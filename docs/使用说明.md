# opeskill 使用说明

## SkillHub 轻量包与全量原子库

SkillHub 单个上传包限制小于 10MB，因此 `opeskill-skillhub.zip` 是轻量包，不直接内置全量大原子库。安装后如需本地离线兜底检索，调用：

```text
$opes-download-atoms
```

或运行：

```powershell
python tools/download_full_atoms.py
```

它会自动从 GitHub Release 下载 `opes-local.zip`，并解压安装 7 位专家原子库和中文商业案例库到：

```text
知识库/原子库/{ahs,dks,hbs,lhs,mrs,nrs,rts}/atoms.jsonl
知识库/商业案例库/atoms.jsonl
```
